# 🎓 Student Grade Calculator

A simple and responsive **Student Grade Calculator** built using **HTML, CSS, and JavaScript**. This web application allows users to enter student details and subject marks, then instantly calculates the total marks, average, percentage, grade, and pass/fail status.

## ✨ Features

- Student Information Form
- Enter marks for 5 subjects
- Input validation (0–100)
- Automatic Total Marks Calculation
- Average Marks Calculation
- Percentage Calculation
- Grade Generation (A+, A, B+, B, C, D, F)
- Pass/Fail Status
- Reset Form
- Responsive Design
- Clean and User-Friendly Interface

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html` or open it in your preferred web browser.
4. Enter the student details and marks.
5. Click **Calculate Result** to view the results.

## 📊 Grade Criteria

| Percentage | Grade |
|------------|-------|
| 90% – 100% | A+ |
| 80% – 89% | A |
| 70% – 79% | B+ |
| 60% – 69% | B |
| 50% – 59% | C |
| 40% – 49% | D |
| Below 40% | F |

> A student must score at least **35 marks in each subject** to be declared **PASS**.
